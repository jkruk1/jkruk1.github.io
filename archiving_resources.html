Video Preservation

youtube-dl

https://pypi.org/project/youtube_dl/#files

Command-line tool for downloading videos directly from YouTube. Hugely adaptable.

C:\Users\space\Desktop>youtube-dl URL

Savefrom

https://en.savefrom.net

Web-based tool for downloading videos from major platforms: YouTube, Vimeo, Yandex, DailyMotion, Facebook, Twitter, etc., etc. Good tool, though it's a bit inconvenient to actually get to the downloading menu. As a shortcut, you can type ssyoutube.com/watch?xyz132 and automatically be taken to the site.

Web Archiving

Wget

Wget 1.20 Manual

GNU Wget is a bare bones program for downloading content off of the Web.

C:\Users\space\Desktop wget URL

For greater utility: wget --recursive --convert-links --html-extension

Archive.today

http://archive.is/

Quick and effective utility. Biggest strength is that it bypasses robots.txt and is able to snap Facebook, Instagram, and many other hard-to-archive sites.

Wayback Machine Archive

http://web.archive.org

The crown jewel of the Internet Archive. While it automatically scrapes a large chunk of the web, you can direct it to manually capture a webpage using web.archive.org/save/*****. Has a moderate number of limitations - robots.txt, pages can opt to be excluded, and it struggles with multimedia - but it works well and is built to last.

Archive Search

Wayback Machine Downloader

https://github.com/hartator/wayback-machine-downloader/blob/master/README.md

An extremely useful tool which allows you to download all pages of a website from the Wayback Machine. After the gem has been installed, navigate to Command Prompt and execute:

wayback_machine_downloader http://example12345.com
